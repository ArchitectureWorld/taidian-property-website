# 泰典物业官网资料需求清单

> 目的：收集足够、真实、可核验的资料，用于把当前视觉原型替换为正式官网内容。

## P0：首次正式发布前必须提供

### 1. 企业身份

| 数据 | 要求 | 用途 |
|---|---|---|
| 公司法定全称 | 营业执照上的完整名称 | 页脚、法律信息、联系页 |
| 对外简称 | 已确认：泰典物业 | 导航、首屏、传播标题 |
| Logo | 优先 SVG；其次透明底 PNG，宽度不少于 1000px | 导航、网站图标、宣传二维码 |
| 成立年份 | 精确到年份 | 经验年限展示 |
| 一句话介绍 | 20—35字 | 首屏与微信分享摘要 |
| 公司简介 | 150字版本、500字版本各一份 | 首页与“关于我们” |
| 核心服务区域 | 城市、区县或主要覆盖范围 | 公司介绍与合作说明 |

### 2. 联系与传播信息

| 数据 | 要求 | 用途 |
|---|---|---|
| 业务联系电话 | 可公开、长期有效 | 顶部导航与联系区 |
| 联系人或部门 | 如“市场合作部” | 商务咨询 |
| 办公地址 | 可公开的完整地址 | 联系页、地图 |
| 业务邮箱 | 企业邮箱优先 | 商务咨询 |
| 服务时间 | 真实工作时间 | 联系区 |
| 微信公众号二维码 | 原始高清图片 | 移动端转化 |
| 企业微信二维码 | 可选 | 一对一咨询 |
| 微信分享封面 | 建议 1200×630px | 微信链接分享卡片 |

### 3. 经验与能力数据

所有数字必须可核验；没有可靠依据的数据宁可不展示。

| 数据 | 建议口径 |
|---|---|
| 物业服务年限 | 从公司或核心团队开始从业的明确口径中选择一种，并统一说明 |
| 当前服务小区数量 | 只统计仍在服务的项目 |
| 累计服务项目数量 | 与“当前服务数量”分开 |
| 服务住户数量 | 说明是当前户数还是累计户数 |
| 员工数量 | 说明统计日期 |
| 老旧小区项目数量 | 先定义“老旧小区”的内部统计标准 |
| 客户满意度 | 必须有真实调查时间、样本和计算口径 |
| 续约率或长期合作年限 | 仅在数据可靠时使用 |
| 荣誉与资质 | 名称、颁发单位、年份、证书扫描件 |

### 4. 代表项目资料

首版建议准备 **3—6个真实住宅项目**，老旧小区项目优先。

每个项目需要：

- 项目正式名称；
- 所在城市和区县；
- 项目类型：普通住宅、老旧小区、旧改后社区等；
- 建成年代；
- 泰典物业进场时间；
- 服务年限；
- 建筑面积、楼栋数、住户数；
- 服务范围；
- 接管前的主要问题；
- 泰典物业采取的主要措施；
- 可以公开的真实改善结果；
- 6—12张现场照片；
- 项目名称和图片是否允许公开。

### 5. 服务内容

需要确认泰典物业真实提供的服务项目，例如：

- 客户服务；
- 秩序维护；
- 环境保洁；
- 绿化养护；
- 公共设施维修；
- 设备巡检；
- 停车秩序；
- 社区沟通；
- 老旧小区专项治理；
- 其他增值服务。

每项服务需要一段 **30—60字的实际做法**，不要只提供口号。

## P0：图片素材

### 首屏主图

优先提供以下类型中的一组：

- 老旧小区中物业人员与居民自然交流；
- 小区树荫、楼栋与生活场景；
- 物业人员在社区内巡查或服务；
- 能体现“真实、温暖、被认真管理”的社区画面。

规格建议：

- 横向照片；
- 原图宽度不低于 3000px；
- 不使用带水印的图片；
- 避免摆拍感过强；
- 确认人物肖像和项目场景具备公开使用授权。

### 案例图片

每个项目至少提供：

- 小区入口；
- 楼栋和公共空间；
- 环境维护；
- 工程维修或巡检；
- 物业人员服务；
- 居民活动或沟通；
- 有真实依据的改善前后对比图。

## P1：第二轮完善资料

- 公司发展历程；
- 核心团队介绍；
- 员工培训与标准化管理资料；
- 客户评价或合作方评价；
- 新闻与社区活动；
- 招聘岗位与人才理念；
- 数字化管理工具的真实界面和使用场景；
- 老旧小区治理方法总结；
- 企业宣传视频。

## 域名与发布资料

- 计划使用的域名；
- 域名当前持有人；
- GitHub 仓库归属账号；
- Vercel 账号归属；
- 是否需要绑定自定义域名；
- 是否需要网站访问统计；
- 是否面向中国大陆居民作为主要受众；
- 后续是否需要国内节点、备案或双平台部署。

## 不应直接提供或公开的数据

- 业主姓名、手机号、房号等个人信息；
- 门禁、监控、消防等敏感点位；
- 未经授权的居民正脸照片；
- 内部报价底价、员工身份证明和工资信息；
- 无法核验的满意度、服务数量或荣誉；
- 与项目合作方存在保密约定的资料。
