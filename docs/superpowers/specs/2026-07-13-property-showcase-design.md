# 物业公司展示型官网 HTML 原型设计

## 目标

制作一个可直接打开和分发的展示型官网原型。访客需要快速记住：公司名称、住宅物业经验丰富、服务具有高性价比。

## 视觉方向

- 真实老旧社区影像作为主体。
- 深蓝黑与炭黑为基调，香槟金作为唯一强调色。
- 采用苹果式大画面、大标题、少文字、章节式滚动叙事。
- 以建筑坐标、细网格、标注线和轮廓线表达克制的科技感。
- 不使用数据卡片墙、图标矩阵、霓虹赛博城市或虚构软件后台。

## 页面章节

1. 首屏：公司名称与业务定位。
2. 经验：以大数字突出长期住宅物业经验。
3. 性价比：通过“更懂问题、减少浪费、服务更稳定”解释高性价比。
4. 老旧社区：强调公司对老旧住宅小区的理解。
5. 案例：提供真实项目内容的替换位置。
6. 联系：电话、微信二维码和合作入口。

## 实现约束

- 使用独立的 HTML、CSS 与原生 JavaScript 文件，职责清晰。
- 图片使用仓库内本地 WebP 资源，离线可预览。
- 移动端优先，微信浏览器可正常阅读。
- 不依赖外部框架或 CDN。
- 支持 `prefers-reduced-motion`。
- 文案和数据均为示例占位，明确提示后续替换。
